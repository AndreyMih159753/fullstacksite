Наши технологии

Для изучения FullStack-разработки мы используем много современных технологий

JavaScript

Основной язык программирования. Буквально логика сайта

TailwindCSS

Язык CSS для Nuxt.JS, чтобы быстро писать CSS-классы в классах элементов

Nuxt.JS

Фреймворк для создания приложений, основанный на Vue.JS

Git

Приложение для контроля версий и связи с GitHub

WebSocket

Протокол для двустороннего обмена данными между клиентом и сервером

OAuth 2.0

Протокол для авторизации через сторонние сервисы
